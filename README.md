# Elfsight Test Task (Rick and Morty API)

## Установка и запуск

### Устанавливать зависимости
```bash
npm install --legacy-peer-deps
```
```bash
npm audit fix
```
без `--force`

### Запуск сайта

```bash
npm run start
```
